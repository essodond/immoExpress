## 🤔 Problèmes?

### Le navigateur ne s'ouvre pas?
→ Allez à: **http://localhost:3000**

### Erreur "Port already in use"?
→ Un autre service utilise le port 3000. Vérifiez les fenêtres ouvertes.

### Les données disparaissent après rechargement?
→ Normal! C'est une démo. Les données sont en mémoire.
