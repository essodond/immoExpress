## 🔄 PASSER AU BACKEND RÉEL

Quand le backend est prêt:

1. Ouvrir: `frontend/src/lib/demoConfig.ts`
2. Changer: `export const DEMO_MODE = false;`
3. Lancer le backend: `cd backend && npm run dev`
4. Redémarrer le frontend
5. C'est tout! 🎉
