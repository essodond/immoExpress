# Contribution au Projet ImmoExpert

## Comment Contribuer

Nous accueillons les contributions! Voici comment participer:

### Avant de Commencer

1. Lisez le `README.md`
2. Vérifiez les issues ouvertes
3. Créez une branche pour votre feature

### Processus de Contribution

1. **Fork** le repository
2. **Clone** votre fork
   ```bash
   git clone https://github.com/yourname/immo.git
   cd immo
   ```

3. **Créez une branche**
   ```bash
   git checkout -b feature/ma-feature
   ```

4. **Faites vos changements**
   - Respectez le style de code existant
   - Testez avant de committer
   - Écrivez des messages de commit clairs

5. **Committez vos changements**
   ```bash
   git commit -m "feat: description courte de votre feature"
   ```

6. **Pushez vers votre fork**
   ```bash
   git push origin feature/ma-feature
   ```

7. **Créez une Pull Request**
   - Décrivez vos changements
   - Liez les issues liées
   - Demandez une review

## Guide de Style

### Code JavaScript/TypeScript
- Utilisez les conventions Next.js
- 2 espaces pour l'indentation
- Noms en camelCase pour les variables
- Noms en PascalCase pour les composants

### Commits
```
feat: Ajoute une nouvelle fonctionnalité
fix: Corrige un bug
docs: Met à jour la documentation
style: Change le formatage
refactor: Refactorise le code
test: Ajoute des tests
```

## Questions?

Ouvrez une issue avec le label `question` ou contactez l'équipe.

## Code of Conduct

- Soyez respectueux
- Pas de harcèlement
- Acceptez les critiques constructives
- Aidez les autres contributeurs

Merci de votre contribution! 🎉
