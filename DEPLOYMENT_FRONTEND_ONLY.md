## 🚀 PROCÉDURE COURTE

### 1. Push le code
```bash
git add .
git commit -m "Deploy"
git push origin main
```

### 2. Render.com
```
Sign Up → New Web Service → Repository: immo
```

### 3. Configuration
```
Build: cd frontend && npm install && npm run build
Start: cd frontend && npm start
Variables: NODE_ENV, NEXT_PUBLIC_API_URL, NEXT_PUBLIC_WHATSAPP_NUMBER
```

### 4. Create & Wait
```
Cliquez "Create Web Service"
Attendez 2-3 minutes
Basta! ✅
```
